LuminaDB

Landing Docs API Account Dashboard

Documentation Overview

LuminaDB combines query execution, storage, cache, transactions, replication, and security in one engine.

Quick Start

Initialize with lumina init, then run lumina start for engine networking.

Data Lifecycle

Use checkpoints and snapshots for persistence, and WAL for durability and recovery.

Security

Account APIs use salted PBKDF2 hashes, minimum-length credentials, and expiring session tokens.

Roblox Integration

Dashboard APIs link Roblox place IDs to universes, generate API keys, and pull live player counts.
