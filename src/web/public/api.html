Landing Docs API Account Dashboard

API Catalog

Production-ready endpoints for system checks and account authentication.

Method Path Description
GET /api/status Returns service health and timestamp.
GET /api/docs Returns available API endpoints.
POST /api/auth/signup Registers a new user account.
POST /api/auth/login Authenticates by username or email and returns a token.
GET /api/dashboard Returns authenticated dashboard metrics.
POST /api/dashboard/api-keys Creates a dashboard API key (returns value once).
POST /api/dashboard/games/link Links a Roblox game by place ID and stores universe metadata.
GET /api/dashboard/games Returns linked games with live Roblox player counts.
